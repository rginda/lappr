# Lappr
A web-based lap timing system that connects to EasyLap hardware via WebHID.

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm start`
4. Open your browser to `http://localhost:8000/`

## Development
This project uses a modern JavaScript toolchain.
- `npm run test` - Run the unit test suite via Vitest
- `npm run lint` - Run ESLint to catch syntax errors and bad practices
- `npm run format` - Format the codebase with Prettier
